AquaRural é uma plataforma open-source para gestão de sistemas comunitários de abastecimento de água em áreas rurais.

O sistema permite registrar unidades consumidoras, hidrômetros e leituras mensais, calcular automaticamente o consumo de água, gerar faturas e acompanhar pagamentos e inadimplência.

Foi projetado para associações comunitárias, cooperativas rurais e pequenos sistemas municipais que precisam de uma solução simples, transparente e auditável para administrar o consumo de água.

Principais funcionalidades:

• Cadastro de consumidores e unidades de consumo  
• Gestão de hidrômetros e histórico de instalação  
• Registro de leituras mensais  
• Cálculo automático de consumo  
• Sistema de tarifas configuráveis  
• Geração de faturas  
• Controle de pagamentos e inadimplência  
• Histórico e auditoria de alterações  

Tecnologias previstas:

• FastAPI  
• PostgreSQL  
• SQLAlchemy  
• API REST  
• Dashboard administrativo
